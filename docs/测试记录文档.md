## 四、异常处理场景（6个）

| 序号 | 异常场景 | 处理方式 | 用户提示 |
|:---:|---------|---------|---------|
| 1 | API调用超时 | `requests.exceptions.Timeout` | ⏰ AI 响应超时，请稍后再试 |
| 2 | 网络连接失败 | `requests.exceptions.ConnectionError` | 📡 网络连接失败，请检查网络 |
| 3 | API Key失效 | 状态码 401 | 🔑 API Key 失效，请重新配置 |
| 4 | 数据文件缺失 | `Path("data").glob("*.md")` 返回空 | ⚠️ 数据文件缺失，请联系老师补齐 |
| 5 | 空输入 | `not question.strip()` | 请输入有效的问题 |
| 6 | API返回格式异常 | `KeyError` / `IndexError` | ⚠️ AI 返回格式异常，请重试 |
