# 小航 · 郑州航院校园信息查询 AI 助手

> 郑州航空工业管理学院人工智能专业大一认知实习项目

---

## 项目简介

"小航"是一款基于 Streamlit + 硅基流动 API 构建的校园智能问答应用，专为郑州航空工业管理学院师生设计。

### 核心功能

- **智能问答**：基于大语言模型，回答校园相关问题
- **身份分流**：支持三类用户身份（新生/在校生/教师），提供差异化回答风格
- **多轮对话**：支持连续追问，保持上下文理解
- **会话历史**：自动保存问答记录，支持导出和清空
- **推荐问题**：每类身份显示4个高频推荐问题，降低使用门槛
- **电话黄页**：独立静态页面，API挂了也能用

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
# 复制示例配置文件
cp .env.example .env

# 编辑 .env 文件，填写你的 API Key
```

### 3. 运行应用

```bash
streamlit run src/app.py
```

## 防幻觉硬规则

1. 只能根据【学校资料】回答，资料里没有的明说"我没收录"
2. 严禁编造电话号码、地址、办公时间、学费金额、人名
3. 涉及金钱/转账，无条件提示防骗提醒
4. 涉及心理危机，立即给援助热线
5. 不接入学校系统，不查个人隐私数据
6. 回答末尾标注 [来源:文件名]
